# Distributed Systems

This directory includes assignments related to **distributed systems concepts and architectures**.

The focus is on understanding how independent systems communicate, coordinate, and handle concurrency and failures.

---

## Topics Covered
- Client-server architectures
- Communication protocols
- Concurrency and synchronization

---

## Learning Objectives
- Understand the challenges of distributed computing
- Design simple distributed applications
- Analyze synchronization and communication problems
- Gain familiarity with real-world distributed system concepts

---

## Languages & Tools
- Java

---

Each assignment is stored in a separate folder with code and documentation explaining the system design and behavior.
