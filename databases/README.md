# Databases

This section contains coursework related to **database systems**, focusing mainly on relational databases and SQL.

The projects explore how data is modeled, stored, queried, and maintained efficiently.

---

## Topics Covered
- Entity-Relationship (ER) modeling
- Relational schema design
- Normalization
- SQL queries
- Constraints and indexes

---

## Learning Objectives
- Design relational databases from real-world scenarios
- Write correct and efficient SQL queries
- Understand database normalization and integrity constraints
- Apply database concepts to practical problems

---

## Technologies & Tools
- SQL
- PostgreSQL
- Database design tools

---

Each project directory may include SQL scripts, database schemas, and short documentation describing the design decisions.
